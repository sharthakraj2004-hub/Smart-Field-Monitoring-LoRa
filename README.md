# Smart Field Monitoring System using LoRa

Wireless monitoring system developed using Arduino Mega, ESP8266, LoRa RA-02, DS18B20 temperature sensor, soil moisture sensor, and ultrasonic water-level sensing.

## Features

* Long-range LoRa communication
* Environmental monitoring
* ESP8266 web dashboard
* Real-time sensor updates
